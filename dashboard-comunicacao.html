Camada Model

A camada Model encapsula o acesso à API. A função fetchWithTimeout() realiza requisições GET com timeout de 8s. Em caso de falha HTTP (404, 500) ou rede, lança exceções capturadas pelo Controller.

fetch() API Promise AbortController try/catch
Tratamento de Falhas
⏱ Timeout: Requisições abortadas após 8 segundos via AbortController
❌ HTTP Errors: Status ≠ 200 lança Error com código
🔄 Fallback: UI exibe mensagem amigável sem travar
🛡 try/catch: Cada chamada isolada para falha localizada
